# Kipaji Monorepo

- `apps/api`: NestJS backend API
- `apps/web`: (optional) Next.js admin UI (to be added)
- `packages/shared`: Shared TypeScript utilities and types

## Getting started

1. Install dependencies

```bash
npm install
```

2. Start API (dev)

```bash
npm run dev:api
```

3. Lint & format

```bash
npm run lint
npm run format
```
